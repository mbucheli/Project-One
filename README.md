# Random Chef
Random Chef gives you a random recipe based on the ingredients you input. Recipe will include instructions, ingredients, calories, and health labels.

## Technology Used
* HTML
* CSS3
* Javascript
* jQuery
* Edamam API
* AJAX
* Google Fonts
